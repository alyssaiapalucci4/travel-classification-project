# Travel Classification Project

## Overview
This project predicts travel cost categories using machine learning classification models.

## Dataset
Travel details dataset including accommodation cost, transportation cost, age, and demographics.

## Models Used
- Logistic Regression
- Random Forest
- Support Vector Machine
- K-Nearest Neighbors

## How to Run
1. Open Google Colab
2. Upload dataset
3. Run notebook from top to bottom

## Requirements
- Python 3
- pandas
- scikit-learn
- seaborn
- matplotlib
